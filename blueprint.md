# 팀 편성 어플리케이션 (Team Setter) 블루프린트

## 목적 및 기능
이 어플리케이션은 팀 구성을 쉽고 즐겁게 도와주는 웹 기반 인터페이스를 제공합니다. 사용자는 팀원 목록을 입력하고, 원하는 팀 수를 지정하며, 특정 인원을 팀장으로 설정할 수 있습니다. 어플리케이션은 팀장을 각 팀에 우선 배치한 후 남은 팀원들을 무작위로 분배하여 시각적으로 몰입감 있는 결과를 보여줍니다.

## 프로젝트 개요
### 시각적 몰입형 디자인 버전 (Alice in Wonderland Storybook)
-   **디자인 컨셉:** '이상한 나라의 앨리스' 스타일의 동화책 일러스트레이션. 빈티지하고 마법 같은 분위기.
-   **시각적 요소:**
    -   **배경:** 4개의 왕국(하트, 다이아몬드, 스페이드, 클로버)이 연결된 판타지 지도 느낌의 배경. 부드러운 텍스처와 비비드한 컬러.
    -   **컴포넌트:** 타로 카드 프레임, 화려한 테두리(Ornate Borders), 종이 질감의 카드 레이아웃.
    -   **아이콘/배지:** 각 팀의 상징(하트-왕관, 스페이드-방패, 다이아-보석, 클로버-덩굴)을 정교하게 디자인.
    -   **타이포그래피:** 고전 동화책에 어울리는 우아하고 장식적인 폰트 스타일.
-   **기술 스택:**
    -   **HTML5 & CSS3:** 고퀄리티 배경 이미지 활용, CSS Filter 및 장식적 보더 구현.
    -   **JavaScript:** 팀별 고유 캐릭터 카드 생성 및 결과 애니메이션.
-   **현지화:** 전체 인터페이스 한글화.

## 팀별 테마 정의
1.  **Team Heart (하트 팀):** 빨간색 테마, 여왕과 홍학, 장미. 타로 카드 디자인.
2.  **Team Spade (스페이드 팀):** 파란색 테마, 체스 기사와 성벽, 방패. 타로 카드 디자인.
3.  **Team Diamond (다이아 팀):** 금색/노란색 테마, 보석과 금광, 반짝이는 효과. 타로 카드 디자인.
4.  **Team Clover (클로버 팀):** 초록색 테마, 숲의 엘프와 거대 버섯, 덩굴. 타로 카드 디자인.

## 변경 이력
-   **v1.0:** 기본 기능 구현.
-   **v2.0:** 다크 네온 테마 적용 (폐기).
-   **v3.0 (Current):** 'Alice in Wonderland' 동화책 스타일 UI/UX 대규모 개편.

## 향후 계획
1.  **배경 일러스트 적용:** 판타지 지도 느낌의 고해상도 배경 적용.
2.  **타로 카드 시스템:** 결과 노출 시 실제 타로 카드가 뒤집히는 듯한 효과 구현.
3.  **팀별 고유 사운드:** 팀 편성 결과 확인 시 각 테마에 어울리는 사운드 추가 검토.

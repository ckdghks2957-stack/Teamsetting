# 팀 편성 어플리케이션 (Team Setter) 블루프린트

## 목적 및 기능
이 어플리케이션은 팀 구성을 쉽고 즐겁게 도와주는 웹 기반 인터페이스를 제공합니다. 사용자는 팀원 목록을 입력하고, 원하는 팀 수를 지정하며, 특정 인원을 팀장으로 설정할 수 있습니다. 어플리케이션은 팀장을 각 팀에 우선 배치한 후 남은 팀원들을 무작위로 분배하여 시각적으로 몰입감 있는 결과를 보여줍니다.

## 프로젝트 개요
### 시각적 몰입형 디자인 버전 (Dark Neon Glassmorphism)
-   **디자인 컨셉:** 미래지향적이고 게이밍 감성이 물씬 풍기는 'Dark Neon' 스타일.
-   **시각적 요소:**
    -   **배경:** 깊이감 있는 다크 그라데이션 배경에 은은하게 움직이는 오로라 효과.
    -   **컴포넌트:** 어두운 반투명 유리 질감(Glassmorphism)에 네온 컬러(Cyan, Magenta) 보더와 글로우 효과.
    -   **타이포그래피:** 가독성이 뛰어난 굵은 고딕체와 네온 텍스트 효과.
    -   **인터랙션:** 호버 시 강렬한 빛반사 효과, 결과 카드 생성 시 순차적 등장 애니메이션.
-   **기술 스택:**
    -   **HTML5 & CSS3 (Baseline):** CSS Variables, Backdrop-filter, Keyframe Animations.
    -   **JavaScript (ES Modules):** 팀 편성 로직, Canvas 또는 CSS를 활용한 배경 효과.
-   **현지화:** 전체 인터페이스 한글화.

## 변경 이력
-   **v1.0:** 기본 기능 구현 (팀원 입력, 팀 수 설정, 결과 생성).
-   **v2.0 (Current):** UI/UX 대규모 개편. 'Dark Neon' 테마 적용, 인터랙션 강화.

## 향후 계획
1.  **디자인 개편:** 기존 라이트 테마에서 다크 네온 테마로 전면 교체.
2.  **애니메이션 강화:** 결과 발표 시 폭죽 효과 또는 슬롯 머신 효과 추가 고려.
3.  **공유 기능:** 팀 편성 결과를 이미지로 저장하거나 클립보드로 복사하는 기능.
